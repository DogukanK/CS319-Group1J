# CS319-Group1J

<b>Project Name:</b> Classroom Helper<br /> <br />
<b>Project Description:</b> This is a web-based application to help both students and teachers to communicate in an easier and convenient way. They can plan homeworks, projects and exams better. They can access homework or project documentation, deadlines, notes given by the teacher and many other important information about the course. Through a calendar students can easily keep track of important dates and deadlines. Also, students can discuss class related subjects via forum. If they want, people can prefer to send a message to another user privately. This provides easy, fast and secure communication.
